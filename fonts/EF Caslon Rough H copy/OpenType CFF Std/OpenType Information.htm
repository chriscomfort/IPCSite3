Thank you for choosing an OpenType Font from the Elsner+Flake Digital Library!

 

In licensing this OpenType Font, you have chosen to use a modern font format that works on both Macintosh and PC platforms. The »OP« in the font name stands for »OpenType Plain«.

 

Please note that you can use this EF OP fonts on Macintosh OS 9.2 and up, on OS X, and on Windows XP (or on Windows 98 and up with ATM 4.1 installed).

 

To guarantee compatibility on both Mac and PC platforms, this font was manufactured as »Single Font«. The choice of the font must be made via the »Font Menu«. If you choose a style like Italic or Bold through the »Style« Menu, you will get an electronic modification instead of a true design.

 

In existing documents composed with traditional EF PostScript or TrueType Fonts, the fonts will need to be assigned again if you wish to use the new OpenType fonts.

 

This font contains the same character and keyboard layout as the classic EF PostScript and TrueType Fonts (at least where it is technically feasible).

 

OpenType fonts can contain up to 65,000 characters. If this font has been provided with alternative characters, you may be able to type characters that formerly were available only through a font change.

 

 

Vielen Dank, dass Sie sich für einen Font aus der Elsner+Flake Digital Library entschieden haben!

 

Mit der Lizenzierung dieses OpenType-Plain-Fonts haben Sie sich zur Nutzung eines modernen plattformneutralen Font-Formates entschlossen.

 

Bitte beachten Sie, dass folgende Systemvoraussetzungen für den Einsatz von OpenType-Fonts gelten: Macintosh ab OS 9.2 und höher, Windows XP (ab Windows 98 und höher nur mit ATM 4.1).

 

Damit ein plattformübergreifender Austausch von Dokumenten möglich ist, wurde der Font als »Single Font« hergestellt, der über das »Font Menü«, nicht über das Menü »Stil« anzuwählen ist. Bei der Anwahl eines fetten oder kursiven Schriftschnittes über das Menü »Stil« erhalten Sie eine elektronische Modifikation.

 

In existierenden Dokumenten, die bereits mit EF PostScript- oder EF TrueType-Fonts aktiviert wurden, müssen Sie aufgrund der unterschiedlichen Schriftnamen die Fonts neu zuweisen, wenn Sie OpenType-Fonts nutzen wollen.

 

Der Zeichenumfang und die Tastenzuweisung entspricht, soweit technisch möglich oder noch sinnvoll, den von uns angebotenen EF PostScript- und TrueType-Fonts.

 

Da OpenType-Fonts bis zu 65.000 Zeichen enthalten können, besteht die Möglichkeit, dass darüber hinaus alternative Zeichen vorhanden sein können, die in traditionellen PostScript- und TrueType-Fonts nur über Fontwechsel zu erreichen waren.
